\begin{abstract}
We study whether a small causal language model can separate sequence transport from token-specific computation without query--key--value attention. The proposed model uses a shared stack of dilated depthwise causal convolutions for sequence mixing, a tied low-rank object indexed by token identity, and narrow deterministic micro-expert residuals. The resulting decode state has fixed size with respect to generated context length and requires no attention matrix or growing key--value cache. We specify matched controls that isolate the sequence mixer, lexical object, and micro-expert residual at approximately 100M parameters. This draft intentionally withholds headline numerical claims until the preregistered one-billion-token runs and held-out evaluations are complete. The study is designed to report both matched-token language-model quality and hardware-specific training and incremental-decode throughput.
\end{abstract}
